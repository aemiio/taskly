# Taskly 📝

A simple task management app built with **React Native**, **TypeScript**, and **Expo**.

## 🚀 Features
- ✅ Add, edit, and delete tasks
- 📝 Attach notes to tasks
- ⏰ Set one-time and recurring reminders
- 📅 Organize tasks efficiently

## 🛠️ Tech Stack
- **React Native** 
- **TypeScript**
- **Expo**
